Системы версификации данных: git. Интегрированные среды разработки: PyCharm

  ==========  1  ==========

Зарегистрироваться на портале GitHub:

    https://github.com/signup/


Скачать и установить приложение GitHub Desktop:

    https://desktop.github.com/


Скачать и установить приложение PyCharm Community Edition:

    https://www.jetbrains.com/pycharm/download/


В настройках PyCharm (Ctrl+Alt+S) подключите свой аккаунт GitHub (Version Control->GitHub-> + -> Login)


Создать заготовку для проекта:

    1. В PyCharm создайте новый проект. В окне создания проекта:
        1.1. Выберите в локальной файловой системе каталог для расположения проекта
        1.2. Допишите к пути расположения проекта имя нового вложенного каталога — это имя станет именем проекта по умолчанию
        1.3. Создайте новое виртуальное окружение VirtualEnv
        1.4. Запустите создание нового проекта
    2. В каталог проекта добавьте файл .gitignore из архива с данным заданием
    3. Опубликуйте проект на GitHub (VCS->Share on GitHub)
    4. Отправьте преподавателю приглашение к данному репозиторию
       (Repository->Settings->Collaborators->Add people->GennDALF)



  =========================

Ссылка на удалённый репозиторий проекта помещается в текущий каталог репозитория в виде TXT файла с именем, соответствующем номеру задачи.

В сервис Журнал отправляется сообщение о выполнении задания со ссылкой на репозиторий.
